# User-Engagement-Analytics-Dashboard
Data visualization and analysis of YouTube trending videos using Python and Seaborn
## Overview
This project analyzes YouTube trending video data to understand content trends and user engagement patterns using Python.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Dataset
YouTube Trending Videos Dataset (Canada region)

## Analysis Performed
- Data cleaning and preprocessing
- Category-wise trend analysis
- Views vs Likes relationship analysis
- Identification of top trending videos

## Project Structure
- analysis.py / notebook: Core data analysis code
- data/: Dataset used for analysis
- visuals/: Generated plots
- report/: Detailed project report

## Key Insights
- Certain content categories dominate YouTube trending videos
- Views and likes show a strong positive correlation
- Engagement varies across different content categories
